JPG Flash Flag

By junwenZhang

发表于 2015-06-16

相册中的闪关灯的开关值是通过exif parse出来的,Exif 2.1标准中通过判断返回值为0则认为关闭了闪关灯
但是Exif 2.2 标准中规定了有3个值可以认为是关闭了闪关灯，见下图代码
ExifInterface exif = new ExifInterface(mFilePath);
String tmp = exif.getAttribute(ExifInterface.TAG_FLASH);
//Exif 2.2 Specifies the resulting Flash tag values
//0000.H = Flash did not fire
//0010.H = Flash did not fire, compulsory flash mode
//0018.H = Flash did not fire, auto mode
